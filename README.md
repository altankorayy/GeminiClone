# GeminiClone

Gemini AI mobile app. Supports both text and image prompt. Latest Gemini 1.5 Pro and 1.5 Flash modals available.

## Tech Used
* UIKit with no Storyboard
* MVVM Architecture
* Animations and effects
* Haptic feedback
* Custom views
* Dynamic TextView
* Separate custom TableView cells
* Dynamic constraints
* Concurrency
* And more...

## API
* ai.google.dev
